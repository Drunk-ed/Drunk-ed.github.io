Project Overview

Phantom_to_Interbotix_teleop is a ROS 2-based teleoperation framework that enables real-time, intuitive control of an Interbotix RX200 robotic arm using a Geomagic Touch (Phantom) haptic device.

The project bridges human hand motions captured by a force-feedback stylus with precise robotic arm movements, allowing natural manipulation of the robot's end-effector.

Useful for robotics research, prototyping, and educational environments requiring human-in-the-loop control.

Motivation and Objectives

Robotic manipulation requires intuitive control, which is difficult using keyboards or joysticks.

  • Natural teleoperation interface
  • Haptic + ROS 2 integration
  • Real-time end-effector & gripper control
  • Foundation for teleoperation research

System Architecture

ROS 2 publisher–subscriber framework:

  • Geomagic Touch → pose & button data
  • Teleop node → input mapping
  • Interbotix RX200 → motion execution

Loop rate: ~50 Hz for smooth motion.

Key Features

Real-Time Teleoperation

  • Continuous stylus mapping
  • Low-latency response

Multi-Mode Control

  • Cartesian control
  • Joint-level wrist rotation
  • Button-based mode switching

Gripper Control

  • Toggle open/close
  • Interbotix API integration
